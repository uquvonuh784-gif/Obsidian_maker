# 🚀 Продвинутые возможности умных кнопок

В этом руководстве мы разберем «магию» Obsidian Maker — использование динамических переменных `{{note:Путь}}` и `{{ai:промпт}}`. Эти инструменты превращают обычные кнопки в мощные скрипты автоматизации.

---

## 📖 1. Переменная `{{note:Путь}}` (Чтение файлов)

Эта переменная позволяет кнопке «заглянуть» в любую другую заметку вашего хранилища и использовать её текст.

### 🧠 Где это полезно?
- **Сборка отчетов**: Извлечение данных из разных файлов в один документ.
- **Интеллектуальный анализ**: Передача текста из другого файла в AI для анализа, не открывая сам файл.
- **Шаблоны-конструкторы**: Сборка новой заметки из кусочков других файлов.

### 🌟 Пример: Анализ лога задач
Кнопка прочитает файл `Задачи.md` и попросит ИИ составить список того, что нужно сделать сегодня.

```om-button
label: "🗓 План на день из задач"
icon: "calendar-check"
action: "insert-template"
prompt: "Ниже приведен список моих задач из другого файла. Выбери из них 3 самых важных на сегодня и оформи как план.
---
{{note:Задачи}}
---"
color: "success"
```

---

## ⚡ 2. Переменная `{{ai:промпт}}` (ИИ внутри текста)

Эта переменная позволяет вставить «кусочек» ответа ИИ прямо в параметр другой команды (например, в название файла или текст).

### 🧠 Где это полезно?
- **Креативные названия**: Автоматическая генерация имен файлов на лету.
- **Динамический контент**: Вставка цитаты дня или случайного совета прямо в шаблон.

### 🌟 Пример: Генератор идей для постов
Кнопка создаст файл с креативным названием, которое придумает ИИ.

```om-button
label: "💡 Создать черновик поста"
icon: "pen-tool"
action: "create-note"
folder: "Черновики"
filename: "Пост — {{ai:Придумай интригующее название для статьи про продуктивность из 3 слов}}"
content: "# Тема: {{ai:Придумай 3 тезиса для статьи про продуктивность}}"
color: "primary"
```

---

## 🌪 3. Супер-сила: Комбинирование переменных

Вы можете вкладывать одну переменную в другую. Например, попросить ИИ прочитать файл и на основе его содержимого придумать название для нового файла.

### 🌟 Пример: Архивация с саммари
Кнопка читает текущую заметку, делает из неё краткое название и сохраняет в архив.

```om-button
label: "📦 В архив с AI-именем"
icon: "archive"
action: "create-note"
folder: "Архив"
filename: "Архив — {{ai:Придумай короткое название (2-3 слова) для текста: {{note:{{title}}}}}}"
content: "{{note:{{title}}}}"
color: "warning"
```

---

## 🛠 Советы по использованию

1. **Пути к файлам**: Если файл лежит в папке, указывайте полный путь: `{{note:Папка/Файл}}`. Расширение `.md` можно не писать.
2. **Кавычки**: Если ваш промпт или переменная содержат двоеточия или спецсимволы, обязательно оборачивайте параметр в кавычки: `prompt: "текст {{переменная}}"`.
3. **Ошибки**: Если плагин не может найти файл, на месте переменной появится текст `[Ошибка: Файл не найден]`. Проверьте правильность написания имени.
