# 🔄 Инструкция: Настройка совместной работы через GitHub

Эта инструкция поможет вам и вашей жене настроить общее хранилище заметок с автоматической синхронизацией через плагин **Obsidian Git**.

---

## 🏗 Шаг 1: Подготовка на вашем компьютере (Админ)

1.  **Создайте репозиторий**: Зайдите на GitHub и создайте новый **Private** репозиторий (например, `Family-Notes`).
2.  **Добавьте жену**:
    *   В репозитории перейдите в **Settings** -> **Collaborators**.
    *   Нажмите **Add people** и введите её логин или почту.
3.  **Инициализируйте Vault**: Если ваши заметки еще не в Git, откройте терминал в папке с заметками и выполните:
    ```bash
    git init
    git remote add origin https://github.com/ВАШ_ЛОГИН/Family-Notes.git
    git add .
    git commit -m "Initial commit"
    git push -u origin main
    ```

---

## 💻 Шаг 2: Настройка на компьютере жены

1.  **Клонируйте репозиторий**: Откройте терминал на её ПК и скачайте заметки в нужную папку:
    ```bash
    git clone https://github.com/ВАШ_ЛОГИН/Family-Notes.git
    ```
2.  **Откройте в Obsidian**: В программе Obsidian нажмите "Open folder as vault" и выберите скачанную папку.

---

## 🛠 Шаг 3: Настройка плагина Obsidian Git (на обоих ПК)

1.  Установите плагин **Obsidian Git** через `Settings` -> `Community Plugins`.
2.  Зайдите в настройки плагина и установите следующие параметры:
    *   **Vault backup interval (minutes)**: `5` (сохранение каждые 5 минут).
    *   **Auto pull interval (minutes)**: `5` (проверка обновлений каждые 5 минут).
    *   **Push on backup**: Включено (`ON`).
    *   **Pull updates on startup**: Включено (`ON`).
3.  **Conflict resolution**: Выберите `Manual` или `Only one file` (чтобы избежать потери данных при одновременной правке).

---

## 🔄 Шаг 4: Использование в Obsidian Maker

Я добавил кнопку быстрой синхронизации прямо в интерфейс нашего плагина.

1.  Откройте **AI Chat** в правой панели.
2.  В верхней части панели (блок Quick Tools) нажмите на иконку **🔄**.
3.  Плагин сам выполнит команду `Pull, Commit and Push`. Вы увидите уведомление от Obsidian о статусе синхронизации.

---

## 💡 Советы по совместной работе
*   **Не редактируйте**: Старайтесь не менять одну и ту же строчку текста в одной и той же заметке в одно и то же время.
*   **Имена файлов**: Не переименовывайте файлы, которые сейчас открыты у другого человека.
*   **Если что-то сломалось**: Git хранит всю историю. Вы всегда можете "откатиться" к версии, которая была час или день назад.
