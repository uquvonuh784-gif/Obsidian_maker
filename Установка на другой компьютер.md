# 💻 Установка и перенос Obsidian Maker на другой компьютер

Эта инструкция поможет вам быстро развернуть плагин на новом устройстве.

---

## 🚀 Способ 1: Прямое копирование (самый быстрый)
Если вы хотите просто перенести текущую версию плагина без использования Git.

1.  **На старом компьютере**: 
    *   Зайдите в папку вашего хранилища (Vault) Obsidian.
    *   Перейдите по пути: `.obsidian/plugins/`
    *   Скопируйте всю папку `obsidian-maker`.
2.  **На новом компьютере**:
    *   Откройте папку вашего хранилища Obsidian.
    *   Зайдите в `.obsidian/plugins/` (если папки нет — создайте её).
    *   Вставьте скопированную папку `obsidian-maker`.
3.  **В Obsidian**:
    *   Зайдите в **Settings** -> **Community Plugins**.
    *   Включите плагин **Obsidian Maker**.

---

## 🛠 Способ 2: Через GitHub (для разработчика)
Если вы хотите работать с исходным кодом и получать обновления.

1.  Установите **Node.js** и **Git**.
2.  Склонируйте ваш репозиторий:
    ```bash
    git clone https://github.com/uquvonuh784-gif/Obsidian_maker.git
    ```
3.  Установите зависимости и соберите проект:
    ```bash
    npm install
    npm run build
    ```
4.  Скопируйте файлы `main.js`, `manifest.json` и `styles.css` в папку плагинов Obsidian (как описано в способе 1).

---

## 📥 Способ 3: Через плагин BRAT
Если вы хотите устанавливать плагин прямо из GitHub одной кнопкой.

1.  Установите плагин **BRAT** из Community Plugins.
2.  В настройках BRAT нажмите **Add Beta Plugin**.
3.  Вставьте ссылку: `https://github.com/uquvonuh784-gif/Obsidian_maker`
4.  BRAT сам скачает и установит последнюю версию.

---

## ⚠️ Важное замечание по настройкам
Параметры плагина (например, **Groq API Key**) хранятся локально в настройках конкретного Obsidian. 

**После установки на новый компьютер вам нужно будет заново ввести ваш API-ключ в настройках плагина.**
