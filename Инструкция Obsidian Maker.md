# 🧩 Справочник: Как использовать Obsidian Maker

Добро пожаловать в **Obsidian Maker**! Этот плагин объединяет функции умного помощника и конструктора кнопок. В этой заметке мы подробно разберём, как пользоваться функцией "Кнопки-скрипты" и где это может быть полезно.

---

## 🔘 Что такое Кнопки-скрипты?

Плагин позволяет вставлять прямо в текст ваших заметок интерактивные кнопки. Нажатие на такую кнопку может создать новую заметку, дописать текст в существующую, открыть ссылку или выполнить системную команду Obsidian.

Чтобы кнопка появилась, вам нужно создать блок кода с пометкой `om-button` и описать свойства кнопки в формате YAML:

> [!TIP]
> **Продвинутым пользователям**: Обязательно посмотрите [Примеры умных кнопок и динамических переменных](Примеры%20умных%20кнопок.md), чтобы узнать, как использовать `{{note:Path}}` и `{{ai:prompt}}`.

````markdown
```om-button
label: Текст на кнопке
action: тип_действия
```
````

---

## 🎨 Визуальные настройки

Вы можете сделать кнопки красивыми и заметными.

- **`icon`**: Добавляет иконку (используются имена [Lucide Icons](https://lucide.dev/icons/), например `lightbulb`, `check`, `file-plus`).
- **`color`**: Цвет кнопки. Варианты: `default`, `primary`, `success`, `warning`, `danger`.
- **`style`**: Стиль заливки. Варианты: `filled` (сплошной), `outline` (контурный), `ghost` (без фона).

**🌟 Пример оформления:**
````markdown
```om-button
label: Важная кнопка
icon: alert-triangle
color: danger
style: outline
```
````

```om-button
label: Важная кнопка
action: тип_действия
icon: alert-triangle
color: danger
style: outline
```

## ⚡ Доступные действия (`action`) и примеры использования

В Obsidian Maker есть умные **переменные**, которые вы можете использовать в параметрах:
- `{{date}}` — текущая дата (например, 2026-03-07)
- `{{time}}` — текущее время (17:30)
- `{{datetime}}` — дата и время
- `{{title}}` — название заметки, в которой сейчас открыта кнопка

### 1. Дописать текст в файл (`append-to-note`)
Незаметно (или с открытием файла) добавляет текст в конец или начало нужной заметки.

**🧠 Где полезно?** 
- Ведение "Inbox" (входящие идеи). Когда пришла мысль, не нужно искать нужный файл. Нажали кнопку — мысль записалась.
- Трекер привычек (добавление отметки о выполнении в журнал).

**Пример: Кнопка "Записать идею"**
````markdown
```om-button
label: Записать гениальную мысль
icon: lightbulb
action: append-to-note
path: Идеи.md
content: "\n- {{time}}: Идея из файла {{title}}"
color: success
open: true
```
````
*(Параметр `open: true` заставит Obsidian сразу открыть файл "Идеи.md", чтобы вы увидели результат. Без него текст добавится в фоновом режиме).*

### 2. Создание новой заметки (`create-note`)
Создаёт новую заметку в нужной папке, с нужным именем и (опционально) на базе шаблона.

**🧠 Где полезно?**
- Создание ежедневных дневников одним кликом с главной страницы (Дашборда).
- Создание карточки клиента или проекта по шаблону.

**Пример: Создать дневник на сегодня**
````markdown
```om-button
label: 📝 Создать Дневник
action: create-note
folder: Дневники/{{date}}
filename: {{date}} — Заметки дня
content: "# Мой день {{date}}\n\n- "
color: primary
```
````

### 3. Открытие существующей заметки (`open-note`)
Открывает файл по выбранному пути. Может открыть в новой вкладке.

**🧠 Где полезно?**
- Создание панели навигации в заметке (Dashboard). Заменяет стандартные ссылки красивыми кнопками перехода к главным разделам вашего хранилища.

**Пример: Ссылка на список задач**
````markdown
```om-button
label: Открыть To-Do
icon: list-todo
action: open-note
path: Задачи.md
newTab: true
color: primary
style: ghost
```
````

### 4. Выполнение команд Obsidian (`run-command`)
Вызывает любую команду Obsidian (как встроенную, так и от других плагинов).

**🧠 Где полезно?**
- Кнопка для быстрого открытия настроек.
- Запуск других сложных макросов, открытие боковых панелей или переключение тем оформления.

**Пример: Открыть настройки**
````markdown
```om-button
label: Настройки Obsidian
icon: settings
action: run-command
command: app:open-settings
color: default
style: outline
```
````

### 5. AI-помощник (`ai-prompt`)
Отправляет текст вашей заметки встроенному ИИ-ассистенту и выполняет ваш промпт (запрос). 
Результат вставляется обратно в заметку: в начало, в конец, или вместо выделенного текста.

💡 **Важно:** всегда оборачивайте длинный текст промпта в двойные кавычки `""`, чтобы избежать ошибок форматирования.

**🧠 Где полезно?**
- Кнопка "Сделать саммари" или "Сгенерировать теги".
- Автоматический перевод выделенного фрагмента.
- Придумывание идей прямо внутри документа.

**Пример: Сделать саммари заметки и вставить в конец**
````markdown
```om-button
label: "🤖 Сделать саммари"
action: "ai-prompt"
position: "end"
prompt: "Сделай краткое описание из 1-2 предложений для текста выше. Ответь только текстом саммари."
color: "primary"
```
````

### 6. Открытие ссылки в браузере (`open-url`)
Открывает URL в вашем браузере по умолчанию.

**🧠 Где полезно?**
- Удобные ссылки на рабочие сервисы (Jira, Google Docs) прямо из проектной заметки.

**Пример: Открыть репозиторий проекта**
````markdown
```om-button
label: GitHub
icon: github
action: open-url
url: https://github.com
color: default
```
````
